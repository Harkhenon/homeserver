- Ubuntu 20.04 or Later / Debian 9 or Later

## Installation :

Run those commands in console as root or with sudo :

```sh
$> git clone https://github.com/Harkhenon/homeserver.git --branch installation
$> cd homeserver
$> bash ./install
```
